# doge_api

## Description

doge_api is a minimal Ruby wrapper for DogeAPI.

## Installation

Just run `gem install doge_api` and you're all set.

If you want to use this as part of a package, just add:

    gem 'doge_api'

to your Gemfile.

## Usage

Check `test.rb` for some examples.
